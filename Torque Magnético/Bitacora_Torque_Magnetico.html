Precesin: $\Omega=\dfrac{\mu B}{L_s}$.

Conclusiones preliminares

Se realiz un anlisis del movimiento armnico simple (MAS) que describe la esfera con momento magntico cuando se desplaza angularmente de su posicin de equilibrio en un campo magntico uniforme. Para pequeos ngulos, la relacin terica para el periodo es

$$ T^2 = \frac{4\pi^2 I}{\mu\,B}. $$

Para el anlisis, se grafic el cuadrado del periodo $T^2$ (s$^2$) en funcin del inverso del campo magntico $1/B$ (T$^{-1}$).
El campo magntico se obtuvo a partir de la corriente $I$ (A) y la constante de bobinas $k_B = 1.36\times10^{-3}$ T/A como se hizo con anterioridad. Por otro lado, el periodo se calcul a partir del tiempo de $N=20$ oscilaciones, $t_N$, mediante divisin real: $T = t_N/N$.

La incertidumbre de $T^2$ se propag como $\sigma_{T^2}=2T\,\sigma_T$ con $\sigma_T=\Delta t/N$ y $\Delta t=0.01$ s.

Se ajust la recta $T^2 = m\,(1/B) + b$ a los datos experimentales (ajuste lineal ponderado por $\sigma_{T^2}$).

Resultados del ajuste (con tus datos):

  • Pendiente: $m = (2.382 \pm 0.275)\times 10^{-3}$ s$^2$T

Usando $I_{\rm bola}=\tfrac{2}{5} m_{\rm bolita} r_{\rm bolita}^2$ , se obtiene el momento magntico desde la pendiente ($m = 4\pi^2 I_{\rm bola}/\mu \Rightarrow \mu = 4\pi^2 I_{\rm bola}/m$):

  • Momento magntico: $\mu = 0.014 \pm 0.009$ Am$^2$

La incertidumbre de $\mu$ incluye la de la pendiente $m$ y la de $I_{\rm bola}$, propagando $\sigma_{I}^2 = \big(\tfrac{2}{5} r^2\,\sigma_m\big)^2 + \big(\tfrac{4}{5} m\,r\,\sigma_r\big)^2$ (con $\sigma_m=1.0\times10^{-4}$ kg y $\sigma_r=2.5\times10^{-5}$ m si el pie de rey mide dimetro 0.05 mm).

En conjunto, los datos muestran la tendencia lineal esperada ($T^2 \propto 1/B$) y el valor de $\mu$ es inconsistente (dentro de incertidumbres) con el de la actividad 1.

Conclusiones preliminares

Se realiz un anlisis del movimiento de precesin inducido por el torque magntico sobre el dipolo cuando su momento angular no est alineado con el campo externo. La relacin terica usada fue

$$ \Omega = \frac{\mu\,B}{L_s}. $$

Para el anlisis se grafic la frecuencia de precesin (\Omega) (rad/s) frente al campo (B) (T). El campo se obtuvo como ( B = k_B I ) con ( k_B = 1.36\times10^{-3}\ \text{T/A} ) y su incertidumbre como ( \Delta B = k_B \Delta I ). La frecuencia de precesin se calcul desde el perodo medido (T_p): ( \Omega = 2\pi/T_p ) y su incertidumbre como ( \sigma_\Omega = (2\pi/T_p^2),\Delta t ) con (\Delta t = 0.01\ \text{s}).

Se ajust la recta ( \Omega = m,B + b ) a los datos experimentales.

Resultados del ajuste (precesin):

  • Pendiente: ( m = (8.011 \pm 0.53)\times 10^{2}\ \text{rads}^{-1}\text{T}^{-1} )

Para obtener (\mu) se requiere (L_s = I_{\text{bola}}\ \omega_s) con ( I_{\text{bola}} = \tfrac{2}{5} M r^2 ) y (\omega_s = 2\pi f). Usando (M=0.1413\ \text{kg},\ r=0.02405\ \text{m},\ f=3.8\ \text{Hz}):

  • ( I_{\text{bola}} \approx 3.269\times10^{-5}\ \text{kgm}^2 )
  • ( \omega_s \approx 23.876\ \text{rad/s} )
  • ( L_s \approx 7.81\times10^{-4}\ \text{kgm}^2\text{/s} )

Luego, el momento magntico se obtiene de ( \mu = m,L_s ), con incertidumbre ( \sigma_\mu = \sqrt{(L_s,\sigma_m)^2 + (m,\sigma_{L_s})^2} ). Tomando (\sigma_m = 5.3\times10^{1}\ \text{rads}^{-1}\text{T}^{-1}) y (\sigma_{L_s} \approx (4/5),M,\omega_s,r,\sigma_r) con (\sigma_r=2.5\times10^{-5}\ \text{m}):

  • Momento magntico (precesin):
    (\mu = 0.626 \pm 0.041\ \text{Am}^2)

El resultado es consistente con el modelo lineal (\Omega \propto B) y, dentro de las incertidumbres, comparable con los valores estimados en las otras actividades.
